# Mini Shoply / ShoplyAndroid

## תיאור הפרויקט
Mini Shoply היא אפליקציית Android Native שפותחה ב־Kotlin במסגרת פרויקט סיום בקורס פיתוח אפליקציות מובייל.

האפליקציה מאפשרת:
- התחברות משתמשים
- ניהול הרשאות לפי תפקיד (`admin` / `user`)
- צפייה בקטלוג מוצרים
- הוספה והסרה של מוצרים לרשימת קניות
- ניהול מוצרים על ידי מנהל מערכת
- חיפוש, סינון וטעינה מדורגת של נתונים (Pagination)
- צפייה בסטטיסטיקות וגרף
- ניהול פרופיל משתמש
- שימוש במיקום נוכחי (GPS)
- אינטגרציה עם שירות חיצוני

המערכת מהווה בסיס להמשך פיתוח בפרויקט הגמר.

## פיצ'רים שמומשו

### Authentication
- מסך התחברות עם משתמשים קשיחים לבדיקה
- זיהוי משתמש לפי שם משתמש וסיסמה

### Roles
- הפרדה בין `admin` ל־`user`
- אדמין יכול:
    - להוסיף מוצר
    - לערוך מוצר
    - למחוק מוצר
- משתמש רגיל יכול:
    - לצפות בקטלוג
    - להוסיף/להסיר מוצרים מרשימת הקניות

### Session Management
- ניהול session לוקאלי באמצעות `SharedPreferences`
- שמירת:
    - סוג משתמש
    - שם משתמש
    - שם תצוגה
    - נתוני מיקום

### Product Management
- מסך `AdminActivity` לניהול מוצרים
- הוספה / עדכון / מחיקה של מוצרים
- ולידציה בסיסית על שדות

### Catalog and Shopping List
- תצוגת קטלוג מוצרים
- רשימת קניות אישית
- הוספה/הסרה של מוצרים בלחיצה על כפתור

### Search and Filter
- חיפוש לפי שם מוצר
- סינון לפי קטגוריה

### Pagination
- טעינה מדורגת של מוצרים באמצעות כפתור "טען עוד"

### Statistics
- מסך סטטיסטיקות
- הצגת:
    - סך כל המוצרים בקטלוג
    - סך כל המוצרים ברשימת הקניות
    - כמות מוצרים לפי קטגוריה
- גרף עמודות לפי קטגוריות

### User Profile
- מסך פרופיל משתמש
- הצגת:
    - שם משתמש
    - תפקיד
- שמירת שם תצוגה

### GPS
- מסך מיקום
- בקשת הרשאת מיקום
- קבלת Latitude / Longitude
- שמירת המיקום האחרון

### External API
- אינטגרציה עם שירות חיצוני להצגת מידע נוסף לפי מיקום

## ארכיטקטורה כללית
הפרויקט בנוי בצורה התואמת לוגיקת MVC בסיסית:

- **Model**
    - `ShoppingItem`
    - נתונים השמורים ב־SharedPreferences
- **View**
    - קבצי XML
- **Controller**
    - `LoginActivity`
    - `MainActivity`
    - `AdminActivity`
    - `StatisticsActivity`
    - `ProfileActivity`
    - `LocationActivity`

---

## מבנה מסכים עיקרי
- `LoginActivity` – מסך התחברות
- `MainActivity` – קטלוג מוצרים ורשימת קניות
- `AdminActivity` – הוספה/עדכון/מחיקה של מוצרים
- `StatisticsActivity` – נתונים מסכמים + גרף
- `ProfileActivity` – פרופיל משתמש
- `LocationActivity` – מיקום נוכחי
